# School_Classbook

Приложение школьного журнала.
Функции:
  - Создание журналов (класс + предмет)
  - Просмотр, создание, редактирование и удаление оценок.
  - Запись ошибок и действий пользователя в файл.

# Заметки

  - На github русские комментарии отображаются некоррекно из-за кодировки win1251. Для корректного отображения необходимо склонировать репозиторий себе.
